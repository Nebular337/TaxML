# TaxML
A learning log and repo for understanding classifier models using UK tax depreciation rules as a method. 

This repo documents the process of a non-technical UK accountant taking his first steps in learning machine learning. The first vector for this exploration is through the creation of a machine learning categorisation tool for assessing the correct capital allowances treatment. Inspired by the tool created by EY, https://www.youtube.com/watch?v=NMw_eOGCm7Y. 

##Nothing in this repo or any output produced is intended to be relied upon for tax or accounting purposes and is in no way related to or affiliated with any accounting firm.##
